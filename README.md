# 💳 Credit Card Transactions Data Analytics using SAS Studio

An end-to-end "Data Analytics" project that analyzes "26,052 Indian credit card transactions" using "SAS Studio". The project demonstrates data cleaning, preprocessing, SQL-based data analysis, statistical analysis, and automated report generation to extract meaningful insights from transactional data.

---

## 📌 Project Overview

This project analyzes credit card transaction data from across India to uncover spending patterns, transaction trends, and customer behavior using SAS programming.

The project applies data analytics techniques including data preprocessing, aggregation, statistical analysis, cumulative calculations, and SQL querying to solve nine real-world analytical problems.

The final output is an automated Excel report containing analytical results and key insights.

---

## 🎯 Project Objectives

- Import and preprocess raw transaction data
- Perform data cleaning and validation
- Analyze transaction patterns using SAS
- Extract meaningful insights from transactional data
- Solve nine analytical case studies
- Generate automated analytical reports
- Demonstrate SQL and SAS programming skills for data analytics

## 📈 Analytical Questions Solved

1. Top 5 cities with the highest transaction spending.
2. Highest spending month for each card type.
3. Cumulative transaction milestone analysis.
4. City with the lowest Gold Card spending contribution.
5. Highest and lowest expense category by city.
6. Female spending contribution across expense categories.
7. Month-over-Month growth analysis (January 2014).
8. Weekend spending efficiency analysis.
9. Fastest city to reach 500 transactions.
